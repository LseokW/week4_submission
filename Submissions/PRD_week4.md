# PRD: Week 4 — Neural Network Physics Labs (PySide6 GUI)

**버전**: 1.0
**작성일**: 2026-04-07
**작성**: AI (Claude Sonnet 4.6) + 사용자 협업
**대상 수업**: 코딩/ML 입문 과정

---

## 1. 제품 개요

### 1.1 목적

물리 데이터를 Neural Network로 학습하는 4개의 실험 Lab을 하나의 PySide6 데스크톱 GUI 앱으로 제공한다.
학생이 하이퍼파라미터를 직접 조작하고 학습 결과가 실시간으로 변하는 것을 관찰함으로써 ML 핵심 개념을 직관적으로 이해하도록 돕는다.

### 1.2 핵심 가치

- **직접 조작**: 버튼·슬라이더로 파라미터를 바꾸면 결과가 즉시 달라진다
- **시각적 피드백**: 학습 중 Loss 곡선·진행률이 실시간 업데이트된다
- **개념 연결**: 각 Lab이 ML 개념(근사 정리, 과적합, 비선형 회귀)과 물리 현상을 연결한다

## 3. 기능 요구사항

### 3.1 공통 요구사항 (모든 Lab)

| ID | 요구사항 |
|----|----------|
| F-01 | 앱은 단일 창(QMainWindow) + 4개 탭(QTabWidget)으로 구성된다 |
| F-02 | 각 탭은 좌측 컨트롤 패널 + 우측 Matplotlib 캔버스로 분리된다 |
| F-03 | [Train] 버튼 클릭 시 백그라운드 스레드에서 학습을 실행한다 (UI 블로킹 없음) |

| F-05 | 학습 완료 후 결과 플롯이 자동으로 캔버스에 렌더링된다 |

| ID | 요구사항 |
|----|----------|
